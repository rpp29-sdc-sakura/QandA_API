# QandA_API
## An API for the questions and answers widget of the Feta team's FEC Atelier project.

---
<br>

### Steps to Replicate:

#### 1. fork and clone the repo
- git clone https://github.com/rpp29-sdc-sakura/QandA_API.git

#### 2. cd into repo and install
- npm run install

#### 3. start mongo
- npm run mongo-start

#### 4. start server
- npm run start
